# HAR Viewer CLI

## Getting Started

Install the CLI globally:

```
npm install -g har-viewer-cli
```

Then launch a HAR file:

```
hr path/to/file.har
```

Prefer not to install globally? Run it ad hoc:

```
npx har-viewer-cli path/to/file.har
```
